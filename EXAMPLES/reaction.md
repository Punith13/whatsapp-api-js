# Reaction

## Add reaction

```ts
import { Reaction } from "whatsapp-api-js/messages";

const reaction_message = new Reaction("message_id", "👍");
```

## Remove reaction

```ts
import { Reaction } from "whatsapp-api-js/messages";

const reaction_remove_message = new Reaction("message_id");
```
